# 🛡️ SecureScan
### Plateforme d'Analyse de Qualité & Sécurité de Code
**Hackathon IPSSI 2026** · Semaine du 2 au 6 mars

---

## 📌 Présentation du projet

**SecureScan** est une plateforme web qui permet d'analyser automatiquement la sécurité d'un dépôt de code source. Elle orchestre plusieurs outils d'analyse open source, agrège leurs résultats et les présente dans un dashboard clair et exploitable.

> L'utilisateur entre une URL Git ou uploade son code — SecureScan fait le reste.
